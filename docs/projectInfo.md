

터치애드에 대한 소개 
- 신용카드 이용정보와 교통카드 결제정보를 활용한 위치 및 맥락 기반 초개인화 타게팅 광고 플랫폼
“터치애드는 결제와 이동 데이터를 모두 활용하는 국내 유일의 광고 플랫폼입니다.
하루 50만 건 이상의 카드 결제 데이터를 통해 ‘지금 어디서, 누가, 무엇을 소비했는지’를 실시간으로 분석하여 실제 타겟 고객에게 유효한 광고를 집행합니다.
교통카드 태깅 기반의 생활 반경·이동 동선 분석도 함께 이뤄집니다.
단순한 관심이 아닌, 실제 구매력을 가진 타겟을 식별합니다.
시간·장소·상황까지 고려한 정밀 타겟팅이 가능합니다.
또한, 기존 광고 채널 대비 높은 비용 효율성 (가성비)을 가지고 있습니다.”

세일즈 타겟
경쟁 심화 지역 내 의원- 동일 상권 내 경쟁이 치열하여 신규 환자 유치에 어려움을 겪는 의원
비급여 진료 중심 의원 - 비급여 항목 비중이 높아 객단가가 높고, 마케팅 효과를 크게 기대할 수 있는 의원
마케팅 효율 개선 희망 의원 - 기존 마케팅 채널에서 명확한 효과를 보지 못했거나, 더 효율적인 채널을 찾는 의원
데이터 기반 경영 도입 의원 - 감에 의존하는 마케팅에서 벗어나 데이터 분석을 통한 과학적 마케팅을 원하는 의원
신규 개원의 - 개원 초기, 병원 인지도를 높이고 안정적인 환자층을 빠르게 확보해야 하는 의원

성형외과 - 비급여 중심, 높은 마케팅 투자 의지 (기관당 급여비용: 4천 5백만원 (실제 매출은 비급여로 훨씬 높음))
내과 - 만성질환 및 일차 진료 수요가 높음 (기관당 급여비용: 8억 5천만원)

##  기본 요건
1. 한글로 커뮤니케이션
2. touch 폴더 하위에 Kogo SaaS Web(한국글로벌헬스케어협동조합 조합사 특화 서비스) 의 하위 서비스인 터치애드 서비스를 위한 html 프로토타입을 구축
3. css 폴더 하위의 style.css token을 기본으로 사용
4. 1920 PC 버전을 기본으로 720 미디어쿼리 반응형 웹으로 설계
5. 참조: 업무프로세스 (Front)
- 1단계: 가치 제안
- 2단계: 주소 입력 및 즉시 데이터 도출(외부 데이터베이스 인터페이스)
